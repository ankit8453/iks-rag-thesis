# Phase 1 Foundation Notes (Weeks 2–4)

This directory is where I (Ankit) take notes during the foundation
learning phase. The structure mirrors the 40-week plan:

- [`cv/`](cv/) — computer vision (transfer learning, EfficientNet,
  augmentation, Grad-CAM, multi-task heads, class imbalance, evaluation).
- [`rag/`](rag/) — retrieval-augmented generation (dense retrieval,
  BM25, hybrid retrieval, cross-encoder rerank, chunking, Llama
  quantization, RAGAS, prompt engineering).
- [`xai/`](xai/) — explainability (Grad-CAM methods, faithfulness vs
  plausibility, chunk attribution, Captum).
- [`iks/`](iks/) — Indian Knowledge Systems context (Vrikshayurveda,
  Krishi Parashara, Upavanavinoda, Sanskrit caveats, copyright).

## Note template

Each file uses this skeleton (already pre-filled with empty sections —
just open and fill):

```markdown
## Key concepts
## Papers read
## Tricky bits
## Open questions
## Code I want to try
```

## Discipline

- One paper per note. Don't conflate "EfficientNet" and "transfer
  learning" into one wall of text.
- Cite. Bib keys in `literature_tracker.csv` should match what shows
  up under "Papers read".
- Open questions are precious — re-read this dir at the start of every
  week and try to resolve at least one.
