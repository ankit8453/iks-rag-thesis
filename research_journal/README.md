# Research Journal

A working journal for the M.Tech thesis. Three cadences:

- **Daily** (`daily/YYYY-MM-DD.md`) — what did I run, what worked, what
  didn't. Cheap and frequent. Skip days when nothing was tried.
- **Weekly** (`weekly/YYYY-Www.md`) — short summary plus next-week
  plan. Mirrors the structure in [`../progress.md`](../progress.md) but
  in narrative form.
- **Monthly** (`monthly/YYYY-MM.md`) — progress vs the 40-week timeline.
  Honest about what's slipping.

## Daily template

```markdown
# YYYY-MM-DD

## Ran
- ...

## Worked
- ...

## Didn't work
- ...

## Tomorrow
- ...
```

## Weekly template

```markdown
# YYYY-Www (e.g. 2026-W21)

## Summary
A few sentences on the arc of the week.

## What got done
- ...

## What slipped
- ...

## Next week
- ...
```

## Monthly template

```markdown
# YYYY-MM

## Timeline check
- Phase / week according to plan: ...
- Phase / week actually at:        ...
- Gap (if any):                    ...

## Highlights

## Lowlights

## Risk register
- ...
```
